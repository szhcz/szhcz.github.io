1. Adaptive Length Image Tokenization via Recurrent Allocation

Image Generation token 自适应长度 

作者单位：MIT CSAIL

原文链接：Adaptive Length Image Tokenization via Recurrent Allocation

解决的问题：

传统视觉系统对图像无论信息复杂度如何都分配相同长度的表征。

本文方法可以动态调整 token 长度。

方法：

  • 通过递归将2D tokens 提取并压缩为1D latent tokens。
  • 根据 token 重构质量判断是否继续处理特定的 2D token，从而实现对部分 token 的动态暂停处理。
  • 通过从1D latent tokens 反向解码生成2D图像 token，并以 reconstruction loss 为优化目标来改进 token 表征的精度。

2. DreamPolish: Domain Score Distillation With Progressive Geometry Generation

3D生成   几何形状与纹理 

作者单位：Zhipu AI, Peking University, Tsinghua University

原文链接：DreamPolish: Domain Score Distillation With Progressive Geometry Generation

解决的问题：

文本到3D生成中几何和纹理质量的问题。传统的3D生成方法常会遇到几何表面不平滑、伪影较多，以及生成的纹理不够一致或真实的问题。

DreamPolish，一种文本转 3D 生成模型，擅长生成精致的几何形状和高质量纹理。

方法：

  • 在几何构造阶段，利用多种神经表示来增强合成过程的稳定性。
  • 并没有仅仅依赖于新颖的采样视图中的先验视图条件扩散（这通常会导致几何表面中出现不需要的伪影），而是结合了一个额外的法线估计器来完善几何细节，以具有不同视场的视点为条件。
  • 我们建议添加一个仅需要几个训练步骤的表面抛光阶段，这可以有效地细化由于先前阶段的有限指导而产生的伪影，并产生具有更理想几何形状的 3D 对象。
  • 使用预训练的文本到图像模型生成纹理的关键主题是在这些模型的巨大潜在分布中找到一个合适的域，其中包含逼真且一致的渲染。在纹理生成阶段，我们引入了一种新颖的分数蒸馏目标，即域分数蒸馏（DSD），以引导神经表示朝向这样的域。
  • 从文本条件图像生成任务中的无分类器引导（CFG）中汲取灵感，并表明 CFG 和变分分布引导代表了梯度引导中的不同方面，并且都是增强纹理质量的必要领域。

3. INFERENCE OPTIMAL VLMS NEED ONLY ONE VISUAL TOKEN BUT LARGER MODELS

VLM  VLM 加速

作者单位：Carnegie Mellon University, Bosch Center for Artificial Intelligence

原文链接：Inference Optimal VLMs Need Only One Visual Token but Larger Models

解决的问题：

Vision Language Models 在视觉理解和推理任务中表现出色，但推理延迟高的问题限制了其实际应用。推理时的高计算成本主要源自 LLM 处理大量图像输入token的开销。要降低推理成本，通常可以选择缩小LLM的规模或减少图像输入token的数量。结果显示 the largest LLM that fits within the inference budget while minimizing visual token count — often to a single token 可以实现 VLM 中的推理最优行为。目前的研究多集中于适度token压缩（例如 5 – 10×），结果表明，计算最优推理机制需要在更高的 token 压缩比下运行。

4. GarVerseLOD: High-Fidelity 3D Garment Reconstruction from a Single In-the-Wild Image using a Dataset with Levels of Details

3D 服装重建 

作者单位：CUHKSZ

原文链接：GarVerseLOD: High-Fidelity 3D Garment Reconstruction from a Single In-the-Wild Image using a Dataset with Levels of Details

解决的问题：

在未见过的图像中，尤其是包含复杂服装变形和人体姿势的图像中，保持高效的泛化能力。

方法：

  • 提出了GarVerseLOD，一个新颖的数据集和框架。
  • 数据库有：服装样式、局部细节、服装变形。首先从“服装风格数据库”中采样基础服装形状，然后从“局部细节数据库”中采样细节对，并将这些细节应用到基础形状上。最后使用“服装变形数据库”中的变形对，进一步调整服装的形状。
  • 使用了基于条件扩散模型的新标签生成方法。
